# /好學生筆記 — 生圖(Stage 2)

在 `/note` 產好的底稿上,逐頁生出「<身份>」視角的好學生筆記圖。
詳見 [`prompts/image_notes_skill.md`](../../prompts/image_notes_skill.md);6 色/視角規則見 [`prompts/image_notes_design.md`](../../prompts/image_notes_design.md)。

**前提**:已用 `/note` 產過底稿(沒有就先跑 `/note`);Antigravity 內建 Nano Banana。

## 步驟
1. 取得底稿 **slug**(= `/note` 印出的,通常是 `日期_檔名`)與**身份**(例:鋼琴老師)。
2. 產逐頁提示清單:
   ```bash
   python3 scripts/image_notes_session.py notes <slug> --identity "<身份>"
   ```
   → `sessions/<slug>/note/banana_prompts_<身份>.md`(每頁拖哪張 + 通用提示 + 該頁內容)。
3. **逐頁生圖(一次一張、各自獨立)**:對每一頁,
   把 `sessions/<slug>/note/base_pNN.png` 用 Nano Banana `/edit` + 該頁提示 →
   疊「<身份>」視角手寫彩色註解(**保留原文一字不改**)→ 存成同目錄 `pNN.png`。
   - 🚨 **一次只做一頁、各自獨立**:每張都是它自己的 `base_pNN`,流水號正確、不會重複第 1 頁。
4. 完成後提示使用者成品在 `sessions/<slug>/note/pNN.png`。

> 想換職業:同一份底稿,改 `--identity` 再跑步驟 2–3 即可(底稿不用重產)。
